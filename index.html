Pandas is an open-source Python library geared towards complex data analysis. While your typical .py file can handle dealing with numbers and data to a small extent, Python either doesn't support or is very slow at configuring large-scale data. Here's where Pandas comes in: it utilizes numpy and scipy to manipulate data on a macro scale.

How To Use

Pandas is a package that requires installation via pip, a Python package manager. Once you've run "pip install pandas", pandas'll be easily accessible in your Python file through "import pandas".

Types of Data Strutures in Pandas

Series

Series in pandas are one-dimensional objects similar to lists or arrays. They hold indexed items that operations and functions can be applied to.

Series can be manipulated really easily -- you can pick and choose elements to change, get a list of things that meet certain boolean criteria, concatenate series together, and do mathematical operations, and the list goes on.

Contextualized data and things with categories are far easier to visualize with pandas than with generic Python, making it a more user-friendly package.

Constructing and Using Series

To create a series: seriesName = pandas.Series(['a', 'b', 'c'])

Printing seriesName would output: 0 a 1 b 2 c dtype: object

Operations Using Series

Series contain indexed data values. The indices are changeable, for further readability. seriesName.index = ["Letter 1", "Letter 2", "Letter 3"]
Printing seriesName now would return: Letter 1 a Letter 2 b Letter 3 c dtype: object

In addition, mathematical operations can be performed (if possible) on all data values within a Series or a specific index. For example, printing seriesName * 2 would return: Letter 1 zz Letter 2 bb Letter 3 cc dtype: object

Read more about the Series constructor and its attributes here.

DataFrames

DataFrames is a data structure made up of rows and columns, similar to a spreadsheet or a table. It¡¯s pretty much a group of series that share an index.

DataFrames are mostly advantageous in that data from outside sources can easily be read into one in one step. In addition, DataFrames are really good with math operations and slicing, so values are easy to access and change efficiently.

Constructing DataFrames

DataFrames are the most commonly used Pandas object. It takes a bunch of inputs, including arrays, Series, other DataFrames, and tuples. The constructor will take any 2-dimensional object where rows and columns can be clearly identified. Let's create a sample DataFrame using a dictionary:

dict = {"name": ["John", "Bob", "Sara", "Balthazar"], "class": [2011, 2016, 2017, 2020], "grade": [88, 90, 92, 94] } df = pandas.DataFrame(dict, columns=["name", "class", "grade"])

Printing df gives you: name class grade 0 John 2011 88 1 Bob 2016 90 2 Sara 2017 92 3 Balthazar 2020 94

Other Cool Pandas Tricks, Because Pandas!

Besides simply showing and modifying data, Pandas also has a ton of functionality in exporting or reading data easily, which really sets it apart from generic Python procedures and other libraries.
So what else can Pandas do?

  • Read and write data to Excel, so you don't have to wrangle with those cells yourself
  • Read and write data to csvs, so you don't have to get a headache looking at walls of numbers yourself
  • Read and write data to JSON. mongodb, .txt docs, and tons more -- because let's be real, you really don't wanna do that yourself
  • Read data from the clipboard and URLs, because who really needs to actually go through with the effort of downloading a file?
  • Plot and subplot datasets using the plt.hist function
  • Check data, account for gaps, and drop any rows that have empty spaces automatically, or fill them with the column mean
  • Conditional indexing
  • Map functions (no more pesky for loops!)
  • Run regression and correlation numbers on statistics (in one step, as opposed to like, 17.6 in Excel)
